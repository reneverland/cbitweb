# 链智科技研究中心官方网站

香港中文大学(深圳) 区块链与智能科技研究中心官方网站

CUHK-Shenzhen Blockchain and Intelligent Technology Research Center

## 项目简介

本项目是一个基于 Vue 3 + Vite 的现代化官网，采用对话式界面设计，集成了真实的 AI 助手功能。

## ✨ 核心功能

- 🤖 **AI 对话助手** - 真实的API集成，智能回答问题
- 🌍 **中英文切换** - 支持中文/英文界面切换
- 🌓 **深色模式** - 支持浅色/深色主题切换
- 👥 **团队展示** - 展示中心核心成员（双语显示）
- 📅 **ICSF 会议** - 历届智能金融国际会议记录
- 🎯 **项目分类** - 研究项目分类展示
- 💬 **对话式UI** - 创新的聊天气泡式界面
- ✨ **动画效果** - 流畅的过渡和动画效果

## 技术栈

- Vue 3 (Composition API)
- Vite
- Tailwind CSS
- Axios
- RemixIcon
- i18n 国际化

## 安装依赖

```bash
npm install
```

## 运行开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```

## 功能说明

### 语言切换
点击右上角的 "中/EN" 按钮可以在中英文之间切换

### 主题切换
点击太阳/月亮图标可以切换深色/浅色主题

### AI 对话
在底部输入框输入问题，AI 助手会智能回答关于研究中心的问题

### 快捷问题
点击输入框旁边的快捷按钮可以快速提问
